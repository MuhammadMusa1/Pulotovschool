Что такое цикл for

Повторяет блок кода для каждого элемента последовательности

Работает с диапазонами чисел, строками, списками

Автоматически перебирает все элементы

# Общая структура
for элемент in последовательность:
    # действия с элементом

Функция range()

range(n) → числа от 0 до n-1

range(a, b) → числа от a до b-1

range(a, b, step) → числа с шагом step

Может быть с положительным и отрицательным шагом

Примеры циклов

# Вывод чисел
for i in range(1, 6):
    print(i)  # 1 2 3 4 5

# Обратный счет
for i in range(5, 0, -1):
    print(i)  # 5 4 3 2 1

# Перебор строки
for letter in "Сеть":
    print(letter)  # С е т ь

Практические примеры

Пример 1: Анализ текста

message = "Привет! Как дела?"
count = 0
for char in message:
    if char == "!":
        count += 1
print("Восклицаний:", count)

Связь с интернетом

Проверка сообщений на ключевые слова

Анализ данных пакетов

Пример: фильтр сообщений

bad_words = ["оскорбление", "спам"]
message = "Это нормальное сообщение"
for word in bad_words:
    if word in message:
        print("Найдено запрещенное слово!")
        break

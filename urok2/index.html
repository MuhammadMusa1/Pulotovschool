Введение

Компьютеры бывают разных размеров, форм и мощностей — от огромных суперкомпьютеров до карманных смартфонов.

А чтобы они могли выполнять наши задачи эффективно, нам нужно научиться правильно работать с данными и переменными в программах.

Это основа всей информатики и программирования — понимание того, как информация хранится и обрабатывается.

Классы компьютеров

Все компьютеры можно разделить на три основные категории по размеру и производительности:

  • Большие компьютеры — мэйнфреймы и суперкомпьютеры, используемые для сложных вычислений
  • Средние компьютеры — серверы и рабочие станции для бизнеса и профессиональной работы
  • Малые компьютеры — персональные компьютеры, ноутбуки, планшеты и смартфоны для повседневного использования

Большие компьютеры

Большие компьютеры — это мощнейшие вычислительные системы, которые используются в самых ответственных областях.

Они способны выполнять миллиарды операций в секунду и обрабатывать колоссальные объемы данных.

  • Используются в научных исследованиях, банковской сфере, космических программах и медицинских расчетах
  • Способны выполнять миллиарды операций в секунду и обрабатывать колоссальные объемы данных
  • Очень дорогие в производстве и эксплуатации, требуют специальных помещений и охлаждения
Пример: суперкомпьютер Fugaku в Японии — один из самых мощных в мире, способный выполнять более 442 квадриллионов операций в секунду

Малые компьютеры

Малые компьютеры — это устройства, которые мы используем каждый день в учебе, работе и повседневной жизни.

Несмотря на небольшие размеры, современные малые компьютеры обладают огромной вычислительной мощностью.

  • Персональные компьютеры, ноутбуки, планшеты и смартфоны доступны практически каждому человеку
  • Подходят для выполнения повседневных задач: учебы, работы, общения, развлечений и игр

Переменные

Переменная — это как коробка с именем, в которой мы храним информацию для последующего использования в программе.

Каждая переменная имеет имя и содержит определенное значение, которое может меняться в процессе выполнения программы.

15
age
Азиз
name
Примеры переменных:
age = 15
name = "Азиз"
temperature = 36.6

Имя переменной помогает нам понять, какая информация в ней хранится и для чего она используется.

Типы данных (современные)

В современных языках программирования существует несколько основных типов данных:

  • int (integer) — целые числа без дробной части, например: 5, -10, 100
  • float (floating point) — числа с плавающей точкой (дробные), например: 3.14, -0.5, 2.0
  • str (string) — строки текста, заключаются в кавычки, например: "Привет", "Азиз"
  • bool (boolean) — логические значения: True (истина) или False (ложь)
  • list, dict, tuple — сложные структуры для хранения нескольких значений одновременно
# Примеры использования различных типов данных
возраст = 15 # int - целое число
рост = 1.75 # float - дробное число
имя = "Азиз" # str - строка текста
успешный = True # bool - логическое значение
оценки = [4, 5, 5] # list - список чисел
ученик = {"имя": "Азиз", "возраст": 15} # dict - словарь

Типы данных (старые)

В старых языках программирования, таких как Pascal и C, программист должен был заранее указывать тип каждой переменной.

Это называлось "статическая типизация" — тип переменной фиксировался при объявлении и не мог измениться.

  • Integer — целые числа
  • Real — дробные числа
  • Char — один символ
  • Boolean — логические значения (истина/ложь)
Пример объявления переменных в языке Pascal:
var age: Integer;
var price: Real;
var letter: Char;
var is_valid: Boolean;

// После этого можно использовать переменные:
age := 15;
price := 99.99;

После объявления переменной нельзя было присвоить ей значение другого типа — это обеспечивало безопасность, но ограничивало гибкость.

Сравнение старых и современных подходов

Раньше программисты должны были строго указывать тип переменной при её создании.

Современные языки программирования, такие как Python, позволяют менять тип данных переменной динамически.

Старый подход (Pascal)

var x: Integer;
x := 5;
// Попытка присвоить строку вызовет ошибку!
// x := "Привет"; // ОШИБКА!
// Компилятор не позволит!

// Каждая переменная имеет
// фиксированный тип навсегда

Современный подход (Python)

x = 5 # x теперь целое число
print(x) # Вывод: 5
print(type(x)) # <class 'int'>

x = "Привет" # x теперь строка!
print(x) # Вывод: Привет
print(type(x)) # <class 'str'>

# Это работает без ошибок!
# Тип определяется автоматически

Современные языки более гибкие, что делает программирование проще и быстрее, особенно для начинающих.

Семантические переменные

Семантические переменные — это переменные с понятными, осмысленными названиями, которые отражают смысл хранимой информации.

Такие имена делают код легко читаемым и понятным для других программистов и для вас самих в будущем.

✅ Хорошие примеры семантических переменных:
student_age = 15
student_name = "Азиз"
total_score = 95
is_student_enrolled = True
average_temperature = 22.5

Когда вы видите такое имя переменной, сразу понятно, какая информация в ней хранится и зачем она нужна.

Несемантические переменные

Несемантические переменные — это переменные с непонятными, случайными или слишком короткими названиями.

Такие имена затрудняют чтение кода и могут привести к ошибкам при его поддержке и развитии.

❌ Плохие примеры несемантических переменных:
x = 15
a1 = "Азиз"
temp_var = 95
flag = True
data = 22.5

Через неделю после написания такого кода вы можете забыть, что означает каждая переменная, и потратить много времени на его анализ.

Почему важно знать переменные

Понимание переменных и правильное их использование — это основа всего программирования.

  • Хорошие переменные упрощают написание и чтение программ, делая код понятным и доступным
  • Правильно названные переменные делают код самодокументируемым — он объясняет сам себя без дополнительных комментариев
  • Семантические переменные помогают избежать ошибок и значительно упрощают поиск проблем в коде
  • Переменные являются основой всех алгоритмов и вычислений в программировании
  • Правильное использование переменных ускоряет разработку и упрощает командную работу над проектами

Примеры кода

Сравним два подхода к написанию кода: с использованием семантических и несемантических переменных.

Семантический подход

school_name = "Школа №15"
students_count = 30
average_mark = 8.5
is_school_open = True
subjects = ["Математика", "Русский", "Информатика"]

Несемантический подход

a = "Школа №15"
b = 30
c = 8.5
d = True
e = ["Математика", "Русский", "Информатика"]

Какой код легче понять? Какой код проще будет поддерживать через месяц?

Правильное именование переменных — это инвестиция в качество вашего кода и ваше будущее.

Практическая задача

Давайте решим простую практическую задачу, чтобы закрепить знания о переменных.

Задача: создать программу, которая спрашивает имя ученика и его оценку, а затем выводит результат.

# Запрос имени ученика
student_name = input("Введите имя ученика: ")

# Запрос оценки и преобразование в число
student_mark = int(input("Введите оценку по шкале от 1 до 10: "))

# Проверка корректности оценки
if student_mark >= 1 and student_mark <= 10:
    print(student_name, "получил(а) оценку", student_mark)
else:
    print("Ошибка: оценка должна быть от 1 до 10")

Эта программа использует переменные для хранения информации и демонстрирует основные операции ввода-вывода, а также простую проверку данных.

ООП в JavaScript — Просто и Ясно!

Коробка 1. Что такое объект?

Объект — это коробка с вещами. Внутри: свойства (что у него есть) и методы (что он умеет делать).

// Создаём объект "человек" let person = { name: "Али", age: 14, sayHello() { alert("Привет! Меня зовут " + this.name); } }; person.sayHello(); // Вызов метода

Рецепт 2. Класс — шаблон для объектов

Класс — это рецепт. Из одного рецепта можно сделать много одинаковых объектов.

// Класс "Животное" class Animal { constructor(name) { this.name = name; } speak() { return this.name + " издаёт звук!"; } } // Создаём объекты let dog = new Animal("Собака"); let cat = new Animal("Кошка");

Семья 3. Наследование — передаём свойства

Наследование — как в семье: ребёнок получает черты от родителя и добавляет свои.

class Dog extends Animal { constructor(name) { super(name); // Вызываем родителя } bark() { return this.name + " говорит: Гав-гав!"; } } let rex = new Dog("Рекс");

Сейф 4. Инкапсуляция — прячем данные

Инкапсуляция — как сейф: деньги внутри, а открыть можно только через дверь (метод).

class Bank { #balance = 0; // Приватное поле (никто не видит!) deposit(amount) { if (amount > 0) this.#balance += amount; } getBalance() { return "Баланс: " + this.#balance + " сом"; } } let myBank = new Bank(); myBank.deposit(500);

Маски 5. Полиморфизм — один метод, разные действия

Полиморфизм — как маски: один метод speak(), но собака гавкает, а кошка мяукает.

let animals = [ new Dog("Бобик"), new Cat("Мурка") ]; animals.forEach(animal => { console.log(animal.speak()); // Разные звуки! });

Трофей Итог: 4 столпа ООП

Объекты + Классы + Наследование + Инкапсуляция + Полиморфизм = ООП!

Теперь ты можешь создавать умный, чистый и красивый код!

Анонимные функции и переменные в Python

  • Что такое lambda-функции
  • Локальные и глобальные переменные
  • Где и зачем это используется

Анонимные функции (lambda)

  • Короткие функции без имени
  • Используются для простых действий
  • Записываются в одну строку
lambda a, b: a + b

Примеры lambda-функций

# Обычная функция def square(x): return x * x # Та же функция через lambda square = lambda x: x * x print(square(5))

Локальные и глобальные переменные

  • Локальная — работает только внутри функции
  • Глобальная — доступна во всей программе
x = 10 # глобальная переменная def my_func(): x = 5 # локальная переменная print(x) my_func() print(x)

Резюме

  • lambda — короткие функции без имени
  • Локальные переменные живут внутри функций
  • Глобальные переменные доступны везде
  • Используйте глобальные переменные осторожно ⚠️

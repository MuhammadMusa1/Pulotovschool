Структура адреса E-mail

Все адреса имеют одинаковый формат:

имя_пользователя @ домен .зона

Разберем реальный пример:

student2024 @ school .tj

student2024 — логин пользователя

@ — разделитель (читается "at" — "на")

school.tj — доменное имя почтового сервера

Как работает электронная почта?

Процесс отправки письма:

  1. Вы пишете письмо в почтовой программе
  2. Письмо отправляется на SMTP-сервер
  3. Сервер находит почтовый ящик получателя
  4. Письмо сохраняется на сервере получателя
  5. Получатель забирает письмо, когда зайдет в почту

Важно:

Письма хранятся на сервере, а не сразу на вашем компьютере!

Безопасность — это важно!

Пароль:

  • Должен быть сложным (цифры+буквы+символы)
  • Не повторять на разных сайтах
  • Никому не сообщать!

Вложения:

  • Проверять отправителя перед открытием
  • Сканировать антивирусом
  • Не открывать подозрительные файлы

Спам:

  • Не отвечать на спам-письма
  • Не переходить по подозрительным ссылкам
  • Использовать фильтры спама

Вспоминаем функции

Что такое функция?

def приветствие():
    print("Здравствуйте, ученики!")

Проблема: функция всегда говорит одно и то же

Решение: нужно передавать данные в функцию!

Аргументы функций

Аргументы = данные для функции

def приветствие(имя):
    print(f"Здравствуйте, {имя}!")

приветствие("Алишер") # Аргумент "Алишер"
приветствие("Мадина") # Аргумент "Мадина"

Ключевые понятия:

Параметр — переменная в определении функции (имя)

Аргумент — конкретное значение при вызове ("Алишер")

Возвращаемые значения

return — вернуть результат работы

Без return (только печать):

def показать_сумму(a, b):
    print(f"Сумма: {a + b}")

показать_сумму(5, 3) # Напечатает "Сумма: 8"

С return (возвращает значение):

def вычислить_сумму(a, b):
    return a + b

результат = вычислить_сумму(5, 3) # результат = 8
print(результат * 2) # Можно использовать дальше!

Типичные ошибки

Ошибка 1: Неправильное количество аргументов

def информация(имя, возраст):
    print(f"{имя}, {возраст} лет")

информация("Али") # ОШИБКА! Нужно 2 аргумента
информация("Али", 15) # Теперь правильно

Ошибка 2: Путаница между print и return

    result = a + b # Вычислили
    # Забыли return! Функция вернет None

х = расчет(5, 3) # х = None (ничего)

    result = a + b
    return result # Теперь вернет сумму

Числа от 1 до 10
Задача 1
Напишите функцию numbers_1_10(), которая выводит числа от 1 до 10 включительно, каждое с новой строки.
Решение
def numbers_1_10(): for i in range(1, 11): print(i) numbers_1_10()

Нажмите "Выполнить" чтобы увидеть результат
Подсказка
Используйте цикл for с функцией range(). Помните, что range(1, 11) генерирует числа от 1 до 10 включительно.
Код выполнен успешно!

Числа с суммой цифр = 5
Задача 3
Напишите функцию sum_digits_5(), которая выводит все числа от 1 до 100, сумма цифр которых равна 5.
Решение
def sum_digits_5(): for i in range(1, 101): if (i // 10) + (i % 10) == 5: print(i) sum_digits_5()
Результат выполнения
Нажмите "Выполнить" чтобы увидеть результат
Подсказка
Для двузначных чисел: десятки = i // 10, единицы = i % 10. Для однозначных чисел это тоже работает.

Сумма чисел до ввода 0
Задача 1
Напишите функцию sum_until_zero(), которая запрашивает у пользователя числа до тех пор, пока не будет введён 0, затем возвращает сумму всех введённых чисел.
Решение
def sum_until_zero(): total = 0 while True: n = int(input()) if n == 0: break total += n return total print(sum_until_zero())
Результат выполнения
Нажмите "Выполнить" чтобы увидеть результат
Подсказка
Используйте бесконечный цикл while True и оператор break для выхода при вводе 0.
Код выполнен успешно!

Простые числа от 2 до 100

Напишите функцию primes_2_100(), которая выводит все простые числа от 2 до 100 включительно.
Решение
def primes_2_100(): for n in range(2, 101): prime = True for d in range(2, n): if n % d == 0: prime = False break if prime: print(n) primes_2_100()
Результат выполнения
Нажмите "Выполнить" чтобы увидеть результат
Подсказка
Число простое, если оно делится только на 1 и на себя. Проверяйте делители от 2 до n-1.

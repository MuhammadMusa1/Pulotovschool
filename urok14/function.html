Функции в Python

Функция — это именованный блок кода, который выполняет определенную задачу и может быть вызван из других частей программы.

Функции помогают:

  • Структурировать код
  • Избегать повторений
  • Упрощать отладку и тестирование
  • Делать программу более читаемой

Аналогия из жизни

Функция как кухонный прибор: вы загружаете ингредиенты (аргументы), нажимаете кнопку (вызов функции) и получаете результат (возвращаемое значение).

Простейшая функция
def приветствие():
    print("Привет, мир!")

# Вызов функции
приветствие() # Выведет: Привет, мир!

Структура функции

Функция в Python состоит из нескольких ключевых элементов:

  1. Ключевое слово def
  2. Имя функции (должно отражать её назначение)
  3. Круглые скобки с параметрами (могут быть пустыми)
  4. Двоеточие
  5. Тело функции с отступом
  6. Необязательное возвращение значения (return)
Структура функции с объяснением
def название_функции(параметр1, параметр2): # Заголовок функции
    '''Документационная строка (описание функции)'''
    результат = параметр1 + параметр2 # Тело функции
    return результат # Возврат значения

# Вызов функции с аргументами
переменная = название_функции(5, 3)
print(переменная) # Выведет: 8

Параметры и аргументы

Параметры — переменные в определении функции.

Аргументы — фактические значения, передаваемые при вызове функции.

Типы параметров

1. Обязательные (позиционные)

2. Именованные (ключевые)

3. Параметры по умолчанию

4. Произвольное количество аргументов (*args, **kwargs)

Примеры параметров и аргументов
# Обязательные параметры
def сложение(a, b):
    return a + b

# Параметры по умолчанию
def приветствие(имя, текст="Привет"):
    return f"{текст}, {имя}!"

# Вызов функций
результат1 = сложение(5, 3) # 5 и 3 - аргументы
результат2 = приветствие("Анна") # Используется значение по умолчанию
результат3 = приветствие("Иван", "Здравствуй") # Переопределение значения

Возврат значений (return)

Оператор return завершает выполнение функции и возвращает указанное значение.

Если return отсутствует, функция возвращает None.

Особенности return

• Может возвращать любое значение (число, строку, список, другую функцию)

• Может возвращать несколько значений (как кортеж)

• Может использоваться для досрочного выхода из функции

Примеры использования return
# Возврат одного значения
def квадрат(число):
    return число ** 2

# Возврат нескольких значений
def мин_макс(список):
    return min(список), max(список)

# Досрочный выход из функции
def проверка_возраста(возраст):
    if возраст < 0:
        return "Некорректный возраст"
    if возраст >= 18:
        return "Совершеннолетний"
    else:
        return "Несовершеннолетний"

Область видимости переменных

Область видимости определяет, где переменная доступна для использования.

Тип области Описание Пример
Локальная Переменная определена внутри функции Доступна только внутри этой функции
Глобальная Переменная определена вне всех функций Доступна во всем коде
Нелокальная Переменная из внешней функции (вложенные функции) Доступна во внутренней функции
Пример областей видимости
глобальная_переменная = "Я глобальная" # Глобальная область

def моя_функция():
    локальная_переменная = "Я локальная" # Локальная область
    print(локальная_переменная) # Работает
    print(глобальная_переменная) # Тоже работает

моя_функция()
# print(локальная_переменная) # ОШИБКА! Переменная не определена
print(глобальная_переменная) # Работает

Встроенные и пользовательские функции

В Python есть два основных типа функций:

Встроенные функции (Built-in)

Уже определены в Python, доступны без импорта.

Примеры: print(), len(), input(), type(), int(), str()

Пользовательские функции

Создаются программистом для решения конкретных задач.

Определяются с помощью ключевого слова def.

Сравнение встроенных и пользовательских функций
# Использование встроенных функций
длина = len("Привет") # Встроенная функция
число = int("123") # Встроенная функция

# Определение пользовательской функции
def среднее_арифметическое(числа):
    return sum(числа) / len(числа) # sum() и len() - встроенные

# Использование пользовательской функции
результат = среднее_арифметическое([1, 2, 3, 4, 5])
print(f"Среднее: {результат}") # Выведет: Среднее: 3.0

Документирование функций

Docstring — строка документации, которая описывает назначение функции, её параметры и возвращаемое значение.

Располагается сразу после заголовка функции в тройных кавычках.

Зачем нужна документация?

1. Помогает понять, что делает функция

2. Упрощает использование функции другими разработчиками

3. Позволяет генерировать документацию автоматически

4. Помогает при отладке и тестировании

Пример документированной функции
def площадь_прямоугольника(длина, ширина):
    '''
    Вычисляет площадь прямоугольника.

    Параметры:
    длина (float): Длина прямоугольника
    ширина (float): Ширина прямоугольника

    Возвращает:
    float: Площадь прямоугольника
    '''
    return длина * ширина

# Получение документации
print(площадь_прямоугольника.__doc__)
# Или с помощью help()
help(площадь_прямоугольника)

Практический пример

Рассмотрим практический пример — программу для работы с оценками студентов.

Программа "Журнал оценок"
def добавить_оценку(журнал, студент, оценка):
    '''Добавляет оценку студента в журнал'''
    if студент not in журнал:
        журнал[студент] = []
    журнал[студент].append(оценка)
    return журнал

def средний_балл(оценки):
    '''Вычисляет средний балл по списку оценок'''
    if not оценки:
        return 0
    return sum(оценки) / len(оценки)

def статистика_группы(журнал):
    '''Выводит статистику по всем студентам'''
    print("=== Статистика группы ===")
    for студент, оценки in журнал.items():
        среднее = средний_балл(оценки)
        print(f"{студент}: {оценки}, средний балл: {среднее:.2f}")

# Использование функций
журнал_оценок = {}
журнал_оценок = добавить_оценку(журнал_оценок, "Иванов", 5)
журнал_оценок = добавить_оценку(журнал_оценок, "Иванов", 4)
журнал_оценок = добавить_оценку(журнал_оценок, "Петрова", 5)
статистика_группы(журнал_оценок)

Итоги и полезные советы

Ключевые моменты:

  1. Функции помогают организовать код и избежать дублирования
  2. Функция определяется с помощью ключевого слова def
  3. Параметры могут быть обязательными, именованными или иметь значения по умолчанию
  4. Для возврата значения используется оператор return
  5. Важно понимать область видимости переменных
  6. Документирование функций (docstring) — хороший тон программирования

Советы начинающим

1. Давайте функциям понятные имена, отражающие их назначение

2. Одна функция должна выполнять одну задачу

3. Не делайте функции слишком длинными (желательно не более 20-30 строк)

4. Всегда тестируйте свои функции с разными аргументами

5. Используйте документационные строки (docstring)

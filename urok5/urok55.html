Что такое компьютерная сеть?

Компьютерная сеть — это соединение компьютеров, которые могут «разговаривать» между собой, передавая данные.

💻
Компьютер 1
📦 Пакет данных
💻
Компьютер 2
🌐 Сеть — это как телефонный разговор между компьютерами!

Как работает передача данных?

1
💻 Один компьютер отправляет данные
2
📦 Данные разбиваются на пакеты
3
🔗 Пакеты проходят по кабелю или Wi-Fi
4
💻 Другой компьютер получает и собирает их обратно
Пример: Представьте, что вы отправляете книгу по почте. Вы не можете отправить всю книгу сразу, поэтому разрываете её на страницы и отправляете каждую страницу в отдельном конверте. На другой стороне страницы собирают обратно в книгу.

Что такое пакет данных?

Это маленькая «часть информации», как письмо в конверте.

📦
Структура пакета
📧 Адрес отправителя
📬 Адрес получателя
📄 Кусочек данных
🔢 Номер пакета
💡 Пакеты могут идти разными путями, но собираются в правильном порядке!

Протоколы — язык общения компьютеров

Чтобы понимать друг друга, компьютеры используют «правила общения» — протоколы.

🌐
HTTP/HTTPS
Для просмотра сайтов в браузере
📧
SMTP
Для отправки электронной почты
📁
FTP
Для передачи файлов между компьютерами

Путь данных в Интернете

1
💻 Ваш компьютер
2
📡 Роутер (Wi-Fi)
3
🏢 Провайдер (компания связи)
4
🌐 Сеть Интернет
5
🖥️ Сервер (компьютер с сайтом)
🔄 Сервер обрабатывает запрос и отправляет ответ назад по тому же пути!

Пример: как открывается сайт

1
⌨️ Вы вводите адрес (google.com)
2
💻 Компьютер отправляет запрос
3
🖥️ Сервер Google отвечает и отправляет страницу
4
🖼️ Ваш браузер показывает её на экране

Арифметические операторы в Python

+
Сложение
Сумма двух чисел
-
Вычитание
Разность двух чисел
*
Умножение
Произведение двух чисел
/
Деление
Частное двух чисел

Дополнительные операторы

//
Целочисленное деление
Результат без дробной части
%
Остаток от деления
Что осталось после деления
**
Возведение в степень
Умножение числа само на себя
Примеры:
10 // 3 = 3 (целая часть от деления)
10 % 3 = 1 (остаток от деления)
2 ** 3 = 8 (2 в степени 3)

Пример программы на Python

# Демонстрация арифметических операторов
a = 15
b = 4

print("Сложение:", a + b) # 19
print("Вычитание:", a - b) # 11
print("Умножение:", a * b) # 60
print("Деление:", a / b) # 3.75
print("Целочисленное деление:", a // b) # 3
print("Остаток от деления:", a % b) # 3
print("Возведение в степень:", a ** b) # 50625

Практическая работа

🎯 Задание: Создать программу-калькулятор
# Программа-калькулятор
print("=== КАЛЬКУЛЯТОР ===")
num1 = float(input("Введите первое число: "))
num2 = float(input("Введите второе число: "))

print(f"{num1} + {num2} = {num1 + num2}")
print(f"{num1} - {num2} = {num1 - num2}")
print(f"{num1} * {num2} = {num1 * num2}")

if num2 != 0:
    print(f"{num1} / {num2} = {num1 / num2}")
    print(f"{num1} // {num2} = {num1 // num2}")
    print(f"{num1} % {num2} = {num1 % num2}")
else:
    print("Деление на ноль невозможно!")

print(f"{num1} ** {num2} = {num1 ** num2}")

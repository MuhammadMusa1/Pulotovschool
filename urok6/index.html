Слайд 3

Что обеспечивает работу сети

  • Компьютер — ваше рабочее место
  • Сетевая карта (NIC) — "уши" компьютера для сети
  • Кабель / Wi-Fi — "дорога" для данных
  • Коммутатор (Switch) — "почтальон" внутри здания
  • Маршрутизатор (Router) — "навигатор" между сетями
  • Интернет-провайдер — "шоссе" в интернет
📡 Каждое устройство имеет свой уникальный адрес (IP-адрес), как домашний адрес!
Слайд 4

Как работает сеть

1 Компьютер разбивает данные на "пакеты"
2 Коммутатор отправляет пакеты нужному получателю в локальной сети
3 Роутер определяет лучший маршрут в другие сети
4 Пакеты собираются обратно у получателя
💡 Сеть — как почтовая служба: письма проходят через промежуточные пункты, но всегда находят адресата!

Слайд 6

Что делает оператор if

Команда if проверяет условие.
Если оно истинно — выполняется действие.

age = int(input("Введите возраст: "))
if age >= 18:
    print("Взрослый")
✅ Если возраст 18 или больше → программа выводит "Взрослый"
Слайд 7

Что делает if / else

else используется, если условие не выполняется.
Это как "в противном случае".

temp = int(input("Введите температуру: "))
if temp > 25:
    print("Тепло")
else:
    print("Прохладно")
🌡️ Программа проверяет: если жарко → говорит "Тепло", иначе → "Прохладно"
Слайд 8

Оператор elif (дополнительные условия)

elif — это "иначе если". Позволяет проверить несколько условий подряд.

grade = int(input("Введите оценку (1–10): "))
if grade >= 9:
    print("Отлично")
elif grade >= 7:
    print("Хорошо")
else:
    print("Нужно подтянуться")
🎯 Программа проверяет условия по порядку и выполняет только первое подходящее!

Уровень 10 баллов (высший)

Мини-проект: система проверки логина и возраста.

name = input("Введите имя: ")
age = int(input("Введите возраст: "))

if name == "Али" and age >= 18:
    print("Привет, Али! Добро пожаловать.")
elif name == "Али" and age < 18:
    print("Али, подрасти немного!")
else:
    print("Привет, гость!")
🔐 Используем логические операторы AND/OR для сложных условий!

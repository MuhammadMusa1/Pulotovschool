Оператор if - основа принятия решений

Как работает if:
Команда if проверяет условие. Если оно истинно — выполняется действие.
Пример из жизни: "Если идет дождь, то возьми зонт"
# Проверка возраста
age = int(input("Введите возраст: "))
if age >= 18:
    print("Взрослый")

Оператор if / else - два варианта

Как работает if/else:
else используется, если условие не выполняется.
Пример из жизни: "Если идет дождь, возьми зонт, иначе надень солнечные очки"
# Проверка температуры
temp = int(input("Введите температуру: "))
if temp > 25:
    print("Тепло")
else:
    print("Прохладно")

Оператор elif - несколько условий

Как работает elif:
elif используется для дополнительных условий между if и else.
Пример из жизни: "Если температура выше 30° - жарко, если от 20° до 30° - тепло, иначе - прохладно"
# Проверка оценки
grade = int(input("Введите оценку (1–10): "))
if grade >= 9:
    print("Отлично")
elif grade >= 7:
    print("Хорошо")
else:
    print("Нужно подтянуться")
